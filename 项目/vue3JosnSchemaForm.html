# 代码格式化工具prettier

vscode安装扩展 Prettier - Code formatter

创建.prettierrc文件

{
    "semi": false,
    "singleQuote": true,
    "arrowParens": "always",
    "trailingComma": "all"
}

vscode设置 setting 搜索 format，选择workspace（只针对此项目设置），将Editor: Format On Save打上勾。此时，项目中会都出一个.vscode的文件夹，里面有针对本项目的setting.json

# vsocde - auto save

vscode-settings搜索 save，选择Files: Auto Save设置为 onWindowChange，一旦我们切换窗口，vscode自动保存，并配合prettier格式化代码。
