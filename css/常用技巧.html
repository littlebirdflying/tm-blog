# 元素占位，不可见

CSS visibility 属性 使元素占位，但不可见

h2 {
  visibility:hidden;
}

# 清除浮动

<div class="container clearfix">
  <div class="floatItem"></div>
</div>

通过CSS伪元素在容器的内部元素之后添加一个看不见的空格“\20”或点“.”，并设置clear属性清除浮动。

zoom: 1 支持 IE6 和 IE7.

.clearfix:after {
  content: '\20';
  display: block;
  height:　0;
  clear:both;
}
.clearfix {
  zoom: 1;
}
